Mayoral election results

Mayoral election rules

Mayoral elections use the supplementary vote system, a type of proportional representation. 

  • This allows for a first and a second choice vote.
  • If no candidate gets half of the first choice votes, all but the top two candidates drop out of the election.
  • The second choice votes from the eliminated candidates are then counted.
